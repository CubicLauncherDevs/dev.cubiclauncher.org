# Introducción

Bienvenido a la documentación oficial de **CubicLauncher**. Aquí encontrarás guías detalladas sobre cómo instalar, configurar y sacar el máximo provecho de nuestro launcher.

## ¿Qué es CubicLauncher?

CubicLauncher es un launcher de Minecraft moderno, diseñado para ser rápido, ligero y altamente personalizable. Nuestra misión es proporcionar una experiencia de juego superior, centrada en la facilidad de uso y el rendimiento óptimo.

## Características Principales

- **Multi-Cuenta:** Soporte para cuentas de Microsoft, Mojang y offline.
- **Gestión de Mods:** Instalador integrado de Fabric, Quilt, Forge y NeoForge.
- **Optimización:** Ajustes automáticos de memoria y Java para mejor rendimiento.
- **Personalización:** Interfaz limpia y moderna con temas personalizables.
