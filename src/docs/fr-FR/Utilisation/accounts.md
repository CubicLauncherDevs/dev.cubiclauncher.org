# Comment utiliser CubicLauncher ?

Beaucoup de gens nous ont demandé comment utiliser notre launcher, ce qui est compréhensible car nous avons une interface assez unique. Voici quelques notions de base plus en détail qui ne sont pas dans le tutoriel.

## Comment ajouter un utilisateur ?

Configurer un utilisateur dans CubicLauncher est très simple, suivez ces étapes :

### Étape 1
Cliquez sur la section utilisateur dans la barre latérale.

<div class="my-4 flex justify-center">
	<img src="https://i.ibb.co/VYdg9N2L/step1.png" alt="Interface du launcher" class="rounded-lg border border-neutral-800 max-w-full" />
</div>

### Étape 2
En cliquant, une fenêtre modale apparaîtra où vous pouvez choisir de créer un compte hors ligne ou de vous connecter avec Microsoft.

<div class="my-4 flex justify-center">
	<img src="/docs/howto/step2.png" alt="easter egg xddd" class="rounded-lg border border-neutral-800 max-w-full" />
</div>

### Étape 3
Selon ce que vous avez choisi, suivez les étapes à l'écran.

Si vous avez choisi hors ligne, une entrée de texte apparaîtra où vous pourrez entrer votre nom d'utilisateur.
<div class="my-4 flex justify-center">
	<img src="/docs/howto/step3.png" alt=">_<" class="rounded-lg border border-neutral-800 max-w-full" />
</div>

Si vous avez choisi la connexion Microsoft, la fenêtre modale suivante apparaîtra.
<div class="my-4 flex justify-center">
	<img src="/docs/howto/step4.png" alt="A launcher for launch them all." class="rounded-lg border border-neutral-800 max-w-full" />
</div>

> Que dois-je faire ?

Copiez le code qui apparaît dans ce menu et allez sur le site de vérification ```https://www.microsoft.com/link```.
Entrez le code donné par le launcher et connectez-vous normalement. Une fois terminé, le launcher sera connecté avec votre compte.
