# Comment utiliser CubicLauncher ?

Beaucoup de gens nous ont demandé comment utiliser notre launcher, ce qui est compréhensible car nous avons une interface assez unique. Voici quelques notions de base plus en détail qui ne sont pas dans le tutoriel.

## Comment ajouter un utilisateur ?

Configurer un utilisateur dans CubicLauncher est très simple, suivez ces étapes :

### Étape 1
Cliquez sur la section utilisateur dans la barre latérale.

<div class="my-4 flex justify-center">
	<img src="https://i.imgur.com/Ex1yCdz.png" alt="Interface du launcher" class="rounded-lg border border-neutral-800 max-w-full" />
</div>

### Étape 2
En cliquant, une fenêtre modale apparaîtra où vous pouvez choisir de créer un compte hors ligne, de vous connecter avec Microsoft, ou d'utiliser un serveur d'authentification externe (Authlib) comme https://ely.by/.

<div class="my-4 flex justify-center">
	<img src="https://i.imgur.com/z2q1yUh.png" alt="easter egg xddd" class="rounded-lg border border-neutral-800 max-w-full" />
</div>

### Étape 3
Selon ce que vous avez choisi, suivez les étapes à l'écran.

#### Hors ligne
Si vous avez choisi hors ligne, une entrée de texte apparaîtra où vous pourrez entrer votre nom d'utilisateur.
<div class="my-4 flex justify-center">
	<img src="https://i.imgur.com/JawOIiW.png" alt=">_<" class="rounded-lg border border-neutral-800 max-w-full" />
</div>

#### Microsoft
Si vous avez choisi la connexion Microsoft et que vous avez déjà un compte premium dans le launcher, un sélecteur apparaîtra où vous pourrez choisir votre compte existant ou en ajouter un autre.

<div class="my-4 flex justify-center">
	<img src="https://i.imgur.com/Lqph5ft.png" alt="Sélecteur de comptes Microsoft" class="rounded-lg border border-neutral-800 max-w-full" />
</div>

Si vous n'avez pas encore configuré de compte, un écran de connexion Microsoft officiel apparaîtra.

<div class="my-4 flex justify-center">
	<img src="https://i.imgur.com/ZmlohG7.png" alt="Connexion Microsoft" class="rounded-lg border border-neutral-800 max-w-full" />
</div>

#### Authlib (serveurs externes)
Si vous avez choisi un serveur externe (Authlib), vous devrez d'abord sélectionner votre serveur d'authentification. Le plus populaire et recommandé est https://ely.by/.

<div class="my-4 flex justify-center">
	<img src="https://i.imgur.com/5ZS4pdK.png" alt="Sélection du serveur d'auth" class="rounded-lg border border-neutral-800 max-w-full" />
</div>

Après avoir entré le lien de votre serveur, il vous sera demandé de vous connecter avec votre compte serveur. Si vous utilisez https://ely.by/, il vous demandera l'email et le mot de passe que vous utilisez pour enregistrer votre compte.

<div class="my-4 flex justify-center">
	<img src="https://i.imgur.com/p6NHuSx.png" alt="Connexion Auth" class="rounded-lg border border-neutral-800 max-w-full" />
</div>
