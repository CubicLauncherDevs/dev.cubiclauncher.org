# How to install a theme

Installing a theme in CubicLauncher is very simple. Just follow these 3 steps.

## Step 1: Download the theme

Find a theme in the https://www.cubiclauncher.org/themes section of the page or in the official https://github.com/CubicLauncherDevs/Themes repository. Download it as a ZIP or CBTH file.

<div class="my-4 flex justify-center">
	<img src="https://i.imgur.com/YJ9ujlR.png" alt="Download button" class="rounded-lg border border-neutral-800 max-w-full" />
</div>

## Step 2: Drag the theme to the launcher

Drag the downloaded ZIP or CBTH file directly onto the CubicLauncher window. The launcher will install it automatically.

<div class="my-4 flex justify-center">
	<img src="https://i.imgur.com/YJ9ujlR.png" alt="Step2" class="rounded-lg border border-neutral-800 max-w-full" />
</div>

## Step 3: Select the theme

Open CubicLauncher, go to the theme selector and choose the one you just installed.

<div class="my-4 flex justify-center">
	<img src="https://i.imgur.com/NoBw8vS.png" alt="Configure theme in the launcher" class="rounded-lg border border-neutral-800 max-w-full" />
</div>

And that's it! You now have your theme installed.
