# Installation

Installing CubicLauncher is a straightforward process. Follow the instructions for your operating system to get started.

## Prerequisites

Before installing, make sure you have **Java** installed on your system. We recommend the following versions:

- **Java 25:** For Minecraft versions 26.1 and above.
- **Java 21:** For Minecraft versions 1.21 and above.
- **Java 17:** For Minecraft versions 1.17 and above.
- **Java 8:** For Minecraft versions 1.16.5 and below.

## Windows

1. Download the `.exe` or `.msi` installer from our downloads page.
2. Run the file and follow the installer steps.
3. Once finished, CubicLauncher will be ready on your desktop.

## Linux

We offer support for multiple formats on Linux:

- **RPM:** Download, grant execution permissions and open.
- **AppImage:** Download, grant execution permissions and open.
- **Debian/Ubuntu:** Install the `.deb` package using `sudo apt install ./cubiclauncher.deb`.
- **ARCH PKGBUILD:** Local compilation is required. Binaries generated by GitHub CI (Ubuntu) may not be compatible with Arch Linux due to its rolling release model. <a href="/docs/arch" class="text-blue-400 hover:underline">Guide</a>.

## macOS

1. Download the corresponding file (`.dmg`, `x64.dmg`, `.app.tar.gz` or `x64.app.tar.gz`).
2. If you downloaded a `.dmg` or `x64.dmg`, open it and drag **CubicLauncher** to your Applications folder.
3. If you downloaded an `.app.tar.gz` or `x64.app.tar.gz`, extract it and move **CubicLauncher.app** to your Applications folder.
4. The first time you open it, you may need to allow execution from **System Preferences > Security & Privacy**.
