# Support

Need help with CubicLauncher? We have several official channels where you can get assistance, report issues and receive help from both the community and the development team.

## Before requesting support

Before opening a support request, make sure to:

- Check the official documentation.
- Verify that you are using the latest version of CubicLauncher.
- Search if your issue has already been reported.
- Gather relevant logs, screenshots or error messages.

## GitHub Issues

Our primary method for reporting bugs, requesting new features or getting technical support is through **Issues** on the official repository.

### When to use GitHub Issues?

Use GitHub Issues for:

- Reporting bugs.
- Requesting new features.
- Reporting compatibility issues.
- Reporting unexpected behavior.
- Tracking known issues.

### Official Repository

**Repository:** https://github.com/CubicLauncherDevs/CubicLauncher

When creating an issue, try to include:

- CubicLauncher version.
- Operating system used.
- Detailed description of the problem.
- Steps to reproduce the error.
- Relevant screenshots or logs.

## Discord Support

We also offer support through our official Discord server.

**Discord Server:** https://discord.com/invite/7VaqSrPukm

### Public support channel

For general questions and common issues, you can use the public support channel.

This channel is ideal for:

- Installation help.
- Configuration issues.
- General inquiries.
- Quick question resolution.

Both community members and team members can help you.

### Ticket System

If your issue requires personalized attention or involves private information, you can open a support ticket.

We recommend using tickets for:

- Complex issues.
- Individual follow-up.
- Cases requiring sharing private information.
- Advanced support.

Once the ticket is created, a team member will review your case and respond as soon as possible.

## Still need help?

If you cannot find a solution in the documentation, you can contact us through GitHub Issues, the public support channel on Discord or via a support ticket.
